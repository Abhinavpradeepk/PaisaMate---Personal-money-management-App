# 💰 PaisaMate – Personal Money Management App

PaisaMate is a personal finance management mobile application built using Flutter.
 It helps users track income, expenses, and manage EMI/loan reminders with a clean and simple UI.

### 🚀 Features

##### 💰 Add & track income
##### 💸 Add & track expenses
##### 📊 Financial dashboard with summary
##### ⏰ EMI & loan payment reminders
##### 🔔 Local notifications for due dates
##### 🎨 designed in Figma
##### 📱 Smooth and responsive mobile experience
### 🛠️ Tech Stack
##### Flutter
##### Dart
##### Figma (UI Design)
##### Local Database (SQLite)
##### Flutter Local Notifications
